## 💻 Projeto

O **Leva e Traz** é uma plataforma web de caronas inteligentes e acessíveis que conecta passageiros a motoristas de confiança em todo o Brasil. Com uma interface moderna e intuitiva, o usuário pode buscar rotas, reservar vagas e avaliar motoristas com segurança.

> _"Viajar nunca foi tão simples e econômico."_

**Funcionalidades:**

- 🔍 Busca de caronas
- 👤 Cadastro e login
- 📊 Dashboard
